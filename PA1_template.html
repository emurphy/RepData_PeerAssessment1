Imputing missing values

  1. Calculate and report the total number of missing values in the dataset (i.e.the total number of rows with s)
sum(is.na(activities$steps))
## [1] 2304
  1. Fill in missing values using mean per interval
library(plyr)
impute.mean <- function(x) replace(x, is.na(x), mean(x, na.rm = TRUE))
filled_activities <- ddply(activities, ~ interval, transform, steps = impute.mean(steps))
  1. Create a new dataset that is equal to the original dataset but with the missing data filled in.
filled_activities <- filled_activities[order(filled_activities$date, filled_activities$interval), ] #plyr orders by group so we have to reorder
  1. Make a histogram of the total number of steps taken each day and Calculate and report the mean and median total number of steps taken per day.
